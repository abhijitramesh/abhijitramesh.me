Problem

Mifos Mobile CN was consuming a mock data layer — it lacked real authentication and API integration via the Open Banking layer. For GSoC 2020, The Mifos Initiative needed a mentor to guide a student through integrating the Android app with the WSO2 API Gateway, which serves as the secure auth and routing layer for Fineract CN microservices.

Contribution

Served as GSoC 2020 mentor for Garvit Agarwal at The Mifos Initiative. Brought first-hand context to the role — had contributed to the same codebase as a GSoC student in 2019 and knew its architecture deeply. Mentoring included:

  • Weekly strategy meetings to plan implementation approach for the WSO2 integration
  • PR reviews ensuring code quality and architectural alignment with Fineract CN conventions
  • Unblocking the student on technical decisions around the Open Banking API layer
  • Tracking progress against GSoC milestones and ensuring goals were met

Technical details

The student work targeted the WSO2 API Gateway integration — replacing the app's mock data layer with real authenticated API calls through the Open Banking API layer on top of Fineract CN's microservices. Stack: Android (Kotlin/Java), REST APIs, WSO2, Fineract CN.

Outcome

Garvit successfully completed the GSoC project: the WSO2 integration PR landed and all program goals were met. Abhijit was subsequently invited to the GSoC Mentor Summit 2020 — a selective invitation extended to standout mentors from that year's program.
