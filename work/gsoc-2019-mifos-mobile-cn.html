Problem

Fineract CN (Apache's next-generation microservices banking platform) lacked a client-facing Android app. The Mifos Initiative serves financial institutions that extend banking services to unbanked populations in remote areas — often with unreliable internet. A functional, offline-capable mobile client was needed to let customers view accounts, manage deposits, review loans, and track transactions.

Contribution

Accepted as a Google Summer of Code 2019 contributor to The Mifos Initiative (mentors: Rajan Maurya and Abhilash Gunasegaran). Proposed and delivered 12 features across the three-month period:

  • Login & validation (May): LoginActivity with proper error messaging for empty fields and invalid credentials
  • Core account views (June): Customer profile, customer details, loan review page, product catalog, recent transactions
  • Data layer (July): DataManager pattern for Customer and Identifier entities, decoupling API responses from UI
  • Account details (July–Aug): Loan details and deposit details views with tab-based navigation; fixed deposit tab ordering
  • Infrastructure (June): Full AndroidX migration, updating all dependencies for modern Android compatibility
  • Java → Kotlin migration: Led the codebase migration from Java to Kotlin while both codebases had to coexist — required deep knowledge of Java-Kotlin interoperability to get the boundary right
  • Offline-first: Designed account and transaction views to function with cached data in low/no-connectivity environments, since the app targets deployment in remote villages

Technical details

Android development in Kotlin (migrated from Java), integrated with Fineract CN REST APIs via MVP architecture. Implemented DataManager classes for reliable data flow between API layer and UI. Handled async API calls, form validation, and error states. Offline caching ensured usability in areas with unstable networks.

Outcome

Completed GSoC with all 12 proposed features shipped and 15 PRs merged. Produced the first functional version of Mifos Mobile CN, distributed as a downloadable APK. The app gives unbanked clients in remote areas a self-service interface to manage their financial accounts — a meaningful step in Mifos's financial inclusion mission.
